# Shape Swipe

모바일 브라우저에서 즐기는 간단한 도형 던지기 게임입니다. 하단에서 생성되는 도형 토큰의 색을 탭으로 바꾸고 위쪽 슬롯으로 스와이프해서 맞추면 점수와 콤보를 얻습니다. Node.js 서버(`server.js`)를 실행하면 같은 Wi‑Fi에 있는 기기에서 접속해 플레이할 수 있습니다.

## 파일 구성

- `index.html`  
  게임 UI의 뼈대를 담당합니다. HUD(점수/콤보/리셋 버튼), 캔버스 영역, 그리고 간단한 플레이 가이드를 포함합니다.

- `styles.css`  
  전체 레이아웃 및 HUD/캔버스/상태 메시지의 미려한 스타일을 정의합니다. 모바일 화면을 고려해 반응형 너비·높이를 조정합니다.

- `game.js`  
  클라이언트 게임 로직의 핵심으로, 다음 기능을 담고 있습니다.
  - enum 형태로 Shape/Color 타입 정의 및 슬롯/토큰 세부 설정
  - 포인터 입력 처리(탭으로 색 순환, 스와이프로 발사)와 물리 업데이트(속도 감쇠, 벽 반사)
  - 캔버스 렌더링(슬롯 강조, 토큰 그림자, 보너스 색 표시) 및 슬롯 간 빈틈 없는 배치
  - 슬롯 커버리지 70% 이상일 때 판정하여 점수·콤보·상태 메시지 갱신. 콤보는 3초 이내에만 유지되며 3배수마다 추가 점수를 지급합니다.
  - 토큰·슬롯 크기 확대, 60초 라운드 타이머(남은 시간 HUD 표시 및 종료 처리, 결과 다이얼로그/재시작 버튼 포함)
  - 너무 약한 스와이프는 즉시 실패 판정

- `server.js`  
  정적 파일을 제공하는 최소 Node.js HTTP 서버입니다. `node server.js`로 실행하면 `http://localhost:3000` (또는 LAN IP)에서 게임에 접속할 수 있습니다.

## 실행 방법

```bash
npm install   # 필요 시 (외부 의존성 없음)
node server.js
```

이후 같은 네트워크의 모바일 기기에서 `http://<PC IP>:3000`으로 접속해 플레이하세요.
